## 🐛 TROUBLESHOOTING (Quick Fixes)

### "App won't load"
→ Run `npm start` in terminal

### "Data disappeared"
→ Check browser localStorage (F12 key → Application → LocalStorage)

### "Form won't submit"
→ Fill ALL required fields: Name, Gender, Age, Alive status

### "Tree looks broken"
→ Drag nodes to arrange them manually

### "Export not working"
→ Check browser permissions for downloads
